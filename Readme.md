## Задание:
```
Python.
Разработать микро вебсервис отображающий простую статическую страницу с текстом: Hello world.
Никаких ограничений на использование модулей и фреймворков нет, можно использовать любые средства.

Ansible.
Разработать небольшой playbook, который выполняет следующее:
1) Установка docker - последней версии docker-engine

2) Сборка образа контейнера, в который необходимо включить разработанный ранее сервис на питоне. 
    За базовый образ можно взять любой понравившийся.

3) Запустить конейнер из собранного ранее образа.

4) По итогам выполнения playbook'а сервис должен быть доступен по адресу хоста, на котором запущен контейнер, по порту 8081.

```

## Решение:

```
1) Разработана программа с использованием библиотеки фласк который возвращает Hello world..
2) Разработан плейбук с 3-мя ролями.

    1. basic/install-defaul-packages - устанавливает стандартные пакеты для начальной работы.

    2. docker-install - добавляет репозиторий docker-engine в зависимости от версии системы, удаляет все зависимости, 
        устанавливает последнюю стабильную версию docker-ce и docker-containerd, чистит кеш после себя.

    3. Разработанная программа должна быть в своем репозитории, но да бы не городить несколько реализовал
        в рамках одного репозитория вместе с плейбуком. Исходники уложил в < roles/deployer/my_app/files/source_code >.
            1\ Клонируем репозиторий с кодом
            2\ Собираем и пушим в Docker HUB (как улучшение можно будет прируктить теги от гита к тегам от образов)
            3\ Пулим образ себе из Регистри.
            4\ Запускаем контейнер в нужной конфигурации.
```

## Запуск плейбука.
```
Т.к часть серкретов от регистри упаковал в ansible-vault запускать требуется так.
ansible-playbook playbooks/docker_install.yaml --ask-vault-pass
```
